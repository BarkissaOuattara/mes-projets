\# MediShield BF



MediShield BF est une plateforme IA de cybersécurité pour protéger, auditer et anonymiser les données de santé numériques au Burkina Faso.



\## Objectif du projet



Le projet vise à créer une couche de sécurité pour les applications de santé existantes.



Il permet de :

\- recevoir des journaux d’activité ;

\- stocker les logs ;

\- détecter des comportements suspects ;

\- générer des alertes ;

\- anonymiser des données patients fictives ;

\- afficher un tableau de bord de sécurité.



\## Stack technique



\- Backend : Django REST Framework

\- Frontend : React

\- Base de données : SQLite au début, PostgreSQL plus tard

\- IA : Python + Scikit-learn plus tard
