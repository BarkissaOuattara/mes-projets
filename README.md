\# Mes projets logiciels



Ce dépôt regroupe une sélection de mes anciens projets de développement logiciel, réalisés dans le cadre de ma formation, de travaux pratiques, de projets académiques et de projets personnels.



\## Objectif du dépôt



L'objectif de ce dépôt est de centraliser mes projets afin de présenter mon parcours, mes compétences techniques et mon évolution en développement logiciel.



\## Technologies utilisées



Les projets présents dans ce dépôt peuvent inclure notamment :



\* Java / Spring Boot

\* Python / Django

\* JavaScript / TypeScript

\* Angular

\* Flutter

\* Firebase

\* HTML / CSS

\* SQL

\* UML

\* API REST

\* Git / GitHub



\## Organisation



Les projets sont regroupés dans le dossier :



```txt

projets/

```



Chaque sous-dossier correspond à un projet, un exercice ou un travail académique.
